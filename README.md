# alovi-skincare

A premium Dutch skincare e-commerce website featuring Lanèche and Courtin brands.

## Features

- Editorial design with frosted glass elements
- Psychological copywriting for conversions
- Fully responsive layouts
- Smooth scroll animations
- Product detail pages with galleries

## Pages

- **alovi_landing_final.html** - Homepage with brand showcase and products
- **alovi_product.html** - Product detail page

## Tech Stack

- Pure HTML, CSS, and JavaScript
- Google Fonts (Playfair Display + Inter)
- Unsplash images for placeholders
- Intersection Observer API for animations

## Local Development

Open any HTML file directly in your browser.
